# Secure Login System

This project is a secure login system built using Flask.

## Features
- User registration and login
- Password hashing using bcrypt
- SQL injection protection
- Session management
- Logout functionality

## Technologies Used
- Python
- Flask
- SQLite
- bcrypt

## How to Run
1. Install dependencies:
   pip install flask bcrypt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
